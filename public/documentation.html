Documentation

This is a REST API for an application called “myFlix” that interacts with a database that stores data about different movies

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies None A JSON object holding data about all movies in the database.
Get data about a single movie by movie title /movies/[title] None A JSON object holding data about a single movie, containing title, director, release year, genre, description, image path, featured and actor property.

Example: 
      
        {
            Title: "Midsommar",
            Director: {
              Name:"Ari Aster",
              Bio: "Ari Aster is an American film director, screenwriter, and
              producer. He is known for writing and directing the A24
              horror films Hereditary (2018) and Midsommar (2019)",
              Birth: 1986
            },
            ReleaseYear: 2019,
            Genre: {
              Name: "Horror",
              Description: "Horror is a genre of storytelling intended to scare,
               shock, and thrill its audience. Horror can be interpreted in many
                different ways, but there is often a central villain, monster,
                or threat that is often a reflection of the fears being
                experienced by society at the time."
            },
            Description:"A couple travels to Northern Europe to visit a rural
            hometown's fabled Swedish mid-summer festival. What begins as an
            idyllic retreat quickly devolves into an increasingly violent and
            bizarre competition at the hands of a pagan cult.",
            ImagePath: "https://m.media-amazon.com/images/......jpg",
            Featured: true,
            Actors: [
              "Florence Pugh",
              "Jack Reynor",
              "Vilhelm Blomgren"
            ]
          }
Get data about a genre by name /movies/genre/[name] None A JSON object holding data about a genre, containing name and description property.

Example: 
      
        {
          Name: "Horror",
          Description: "Horror is a genre of storytelling intended to scare,
          shock, and thrill its audience. Horror can be interpreted in many
          different ways, but there is often a central villain, monster,
          or threat that is often a reflection of the fears being experienced
          by society at the time."
        }
Get data about a director by name /movies/directors/[name] None A JSON object holding data about a director, containing bio, birth year property.

Example: 
      
        {
          Name:"Ari Aster",
          Bio: "Ari Aster is an American film director, screenwriter, and
          producer. He is known for writing and directing the A24 horror films
          Hereditary (2018) and Midsommar (2019)",
          Birth: 1986
        }
Add a new user /users A JSON object holding data about a new user, structured like: 
        
          {
            Username: "Jane Doe",
            Password: "password123",
            Email: "j.doe@mail.com",
            Birthday: new Date("1991-06-14"),
            FavouriteMovies: []
          }
A JSON object holding data about a new user, including an ID: 
        
          {
            Username: "Jane Doe",
            Password: "password123",
            Email: "j.doe@mail.com",
            Birthday: "1991-06-14",
            FavouriteMovies: [],
            id: 1
          }
Update a user's name by ID /users/[id] 
      
        {
          Username: 'Jane New Name Doe'
        }
A text message indicating the user's name was updated successfully.
Add a user's favourite movie by title /users/[id]/[title] None A text message indicating that a movie has been added to the favourite's list.
Delete a user's favourite movie by title /users/[id]/[title] None A text message indicating whether the movie was successfully removed from the favourite's list.
Delete a user by ID /users/[id] None A text message indicating whether the user was successfully removed.
